# Landing Page - Beatriz Vieira

Site de uma página elegante e estratégico para advogada especialista em Direito Trabalhista Empresarial.

## 🎨 Características

- **Design minimalista** com paleta terrosa (marrom sofisticado e bege claro)
- **Tipografia elegante**: Playfair Display (serifada) para títulos e Lato (sans-serif) para textos
- **Totalmente responsivo** para mobile e desktop
- **Otimizado para conversão** com CTAs estratégicos
- **Animações suaves** e rolagem fluida
- **Acessibilidade** com navegação por teclado

## 📋 Estrutura

- `index.html` - Estrutura HTML semântica
- `styles.css` - Estilos CSS com variáveis customizadas
- `script.js` - Interatividade e animações

## 🚀 Como usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. O site funciona 100% no lado do cliente, sem necessidade de servidor

## 📱 Seções

1. **Hero** - Apresentação pessoal da advogada
2. **Escolha** - Dois botões de chamada para ação
3. **Contrato com Legal Design** - Serviço para prestadores
4. **Diagnóstico 360** - Serviço gratuito para empresas
5. **Sobre** - Informações profissionais
6. **Footer** - Contatos e redes sociais

## ⚙️ Personalização

### Foto profissional

Substitua o placeholder de emoji por uma foto real:
- Na seção `about-section`, substitua a `div.placeholder-image` por uma tag `<img>`
